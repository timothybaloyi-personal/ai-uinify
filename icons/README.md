# Extension Icons

Placeholder icons are needed in the following sizes:

- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)  
- icon128.png (128x128 pixels)

## Creating Icons

You can create icons using:

1. Image editing software (Photoshop, GIMP, Figma, etc.)
2. Online icon generators
3. AI image generation tools

## Recommended Design

- Simple, recognizable symbol (e.g., robot emoji, AI symbol, unified dots)
- High contrast colors
- Clear visibility at small sizes
- Consistent with extension branding

## Temporary Solution

For testing purposes, you can use simple solid color squares or basic shapes. The extension will load without custom icons, using Chrome's default icon placeholder.
